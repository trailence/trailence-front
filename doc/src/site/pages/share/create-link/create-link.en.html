Create a public link

To share a trail via a link accessible to everyone, open the trail menu and select "Create public link".
You can then share it by copying the link or using the displayed QR code.

Once a link exists, you can view it again using the "View public link" menu. When the link is shown, a "Delete" button also allows you to remove it.

A public link allows anyone with the link to access the trail. No Trailence account is required to view it.
The link does not allow modification; people accessing your trail via the link cannot edit or delete it.
