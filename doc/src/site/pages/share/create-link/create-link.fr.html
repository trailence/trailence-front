Créer un lien public

Pour partager un parcours via un lien accessible par tout le monde, allez dans le menu du parcours à partager, et sélectionnez "Créer un lien public".
Vous pourrez alors le partager en le copiant ou via le QR code qui s'affiche.

Une fois qu'un lien existe, vous pouvez le revoir en utilisant le menu "Voir le lien public". Lorsque le lien s'affiche, un bouton "Supprimer" vous permet également de supprimer ce lien.

Un lien public permet à n'importe qui ayant le lien d'accéder au parcours. Il n'est pas nécessaire d'avoir un compte sur Trailence pour le consulter.
Le lien ne permet pas la modification, les personnes accédant à votre parcours via la lien ne pourront donc pas le modifier ni le supprimer.
