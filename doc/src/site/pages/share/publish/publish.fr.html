Publier un parcours

Vous pouvez contribuer à la communauté Trailence en publiant un parcours.
Une fois publié, il sera visible par tout le monde via la fonction de recherche de parcours.

La publication se fait en plusieurs étapes:

  • En allant sur le parcours que vous souhaitez publier, sélectionner "Publier" dans le menu.
  • Une copie du parcours est alors créée, dans les brouillons (vous verrez alors dans le menu principal, une section "Mes publications" avec une liste "Brouillons".
  • Pendant l'étape de brouillon, vous pouvez modifier le parcours afin d'en faire une version qui sera publiée à tout le monde. Pour cela, une check-list vous accompagne pour vous aider à publier un parcours de qualité, et qui respecte les règles.
  • Une fois prêt, et la check-list complétée, vous pouvez soumettre le parcours pour publication, en cliquant sur "Publier". Le parcours passera alors à l'étape "En cours de revue".
  • La revue est effectuée par un modérateur, qui vérifiera la qualité et le respect des règles.
  • Une fois validé par un modérateur, le parcours sera visible par toute la communauté Trailence !
  • Si le modérateur ne valide pas votre parcours, il vous laissera un message d'explication, et le parcours apparaîtra alors dans la section "Changements demandés". Vous pourrez alors améliorer le parcours en fonction du retour qui vous a été fait, puis le soumettre de nouveau pour validation.
  • Une fois publié, vous pouvez voir toutes vos publications dans la section "Publiés".
