Créer un partage

Un partage vous permet d'indiquer une liste d'adresses email, avec qui vous souhaitez partager des parcours.
Les personnes avec qui vous partagez n'ont pas besoin d'avoir déjà un compte sur Trailence:

  • Ces personnes recevront alors un email, avec un lien leur permettant de consulter votre partage.
  • Les personnes ayant déjà un compte, verront automatiquement votre partage apparaître dans leur menu "Partagé avec moi".

Sélection des parcours à partager

Il y a différents moyens de partager des parcours:

  • En sélectionnant une liste de parcours précise (dans la liste des parcours d'une collection), puis en utilisant le menu "Créer un partage avec ces parcours".
    Dans ce cas, seuls les parcours que vous avez sélectionnés seront partagés, aucun nouveau parcours n'apparaîtra. En revanche, si vous supprimez certains de ces parcours de votre collection, ils seront supprimés du partage également.
    Si vous modifiez un parcours partagé, les modifications seront visibles par les personnes ayant accès au partage.
  • Tous les parcours d'une collection: vous pouvez également partager une collection entière, en utilisant le menu de la collection "Créer un partage", puis "Toute la collection".
    Dans ce cas, tout parcours ajouté, modifié, ou supprimé, sera reflété dans le partage. Ainsi, les personnes ayant accès au partage, verront la collection telle qu'elle est sur votre compte.
  • Les parcours ayant certains tags: en utilisant le menu de la collection "Créer un partage", puis "Des tags", vous pouvez sélectionner les tags à partager.
    Dans ce cas, tout parcours de la collection ayant un des tags sélectionné, sera visible par les personnes ayant accès au partage.
    Si vous taguez un nouveau parcours avec l'un des tags sélectionnés, ce parcours apparaîtra automatiquement dans le partage.
    De la même manière, si vous enlevez les tags d'un parcours, il sera enlevé du partage.
    Ce type de partage est très pratique si vous souhaitez par exemple partager les parcours que vous avez fait avec une personne: vous pouvez taguer les parcours avec le nom de la personne, et partager ce tag avec cette personne.

Fonctionnement du partage

Les personnes avec qui vous partagez des parcours, pourront voir ces parcours, mais ne pourront jamais les modifier.
Vous pouvez choisir d'inclure, ou non, les photos dans le partage.
Vous pouvez également à tout moment, ajouter ou supprimer des personnes d'un partage.

Lorsque quelqu'un vous a partagé des parcours, vous pouvez vous retirer du partage, en allant dans le menu du partage et cliquer sur "Supprimer". Dans ce cas le partage est supprimé pour vous, mais existera toujours pour les autres personnes.

Le nombre de personnes dans un même partage est limité, si vous souhaitez partager un parcours avec de nombreuses personnes, utilisez plutôt les liens publics.
