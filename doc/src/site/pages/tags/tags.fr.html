Les tags

Les tags vous permettent d'organiser vos parcours au sein d'une collection.
Par exemple, vous pouvez créer des tags:

  • Avec le nom des personnes qui étaient avec vous → permet de facilement retrouver les parcours avec une personne, et de partager ces parcours avec cette personne
  • "bivouac" → retrouvez tous vos parcours de bivouac
  • etc...

Un parcours peut avoir plusieurs tags.
