Importer des parcours

Pour importer des parcours depuis un fichier, allez dans une collection, puis utilisez le menu "Importer".
Puis sélectionnez un fichier sur votre appareil.

Les formats supportés sont:

  • Le format GPX : format standard de partage d'une trace GPS, vous trouverez souvent sur des sites ou applications de trace GPS une fonction pour exporter ou télécharger au format GPX.
  • Fichier ZIP contenant des GPX, tous les GPX contenus seront alors importés.

Bien sûr, un fichier exporté depuis Trailence peut être ensuite importé dans Trailence. Dans ce cas, selon les options choisies au moment de l'export, les parcours importés pourront contenir plus d'information qu'un fichier GPX standard (comme les photos, les tags, ...).
