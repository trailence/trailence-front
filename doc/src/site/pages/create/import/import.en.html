Import trails

To import trails from a file, go to a collection, then use the "Import" menu.
Then select a file on your device.

Supported formats are:

  • The GPX format: the standard format for sharing a GPS track; many GPS sites and apps offer an export or download in GPX format.
  • ZIP file containing GPX files: all GPX files contained will be imported.

Of course, a file exported from Trailence can be re-imported into Trailence. In that case, depending on the export options chosen, the imported trails may contain more information than a standard GPX file (such as photos, tags, ...).
