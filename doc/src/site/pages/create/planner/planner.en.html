Route planner

The "Plan a trail" menu lets you create a new trail manually.

  • Start by navigating the map to the starting location. You can use the "Search for a place" function to move quickly to that location.
  • Then click "Start"

Using OpenStreetMap paths

When you begin, all known paths from OpenStreetMap are shown in blue.
This lets you quickly follow these paths to build your trail:

  • Place the first point by clicking the start location
  • Once the start point is set, only the paths that pass through the last point remain highlighted in blue
  • Click a point on a blue path to add a segment to that point to your trail.
  • If you want to take a different branch, click the junction point between the current path and the path to take, so you can continue the trail on the new path

Placing free points

OpenStreetMap paths make it easy and quick to build a trail using known paths.
But sometimes the path you want may be missing, or a connection may be absent. You can then place free points on the map using the "Free point" menu.
You can then switch back to "Follow paths" mode using that menu.

Save the trail

When your trail is finished, click "Save", enter the trail name and the collection where you want to save it.
