Planificateur de parcours

Le menu "Planifier un parcours" vous permet de créer un nouveau parcours manuellement.

  • Commencez par naviguer sur la carte pour aller jusqu'au lieu de départ. Vous pouvez utiliser la fonction "Chercher un lieu" pour vous déplacer rapidement vers ce lieu.
  • Cliquez ensuite sur "Commencer"

Utiliser les chemins Open Street Map

Lorsque vous commencez, tous les chemins connus sur Open Street Map s'affichent alors en bleu.
Cela vous permet de suivre rapidement ces chemins pour créer votre parcours:

  • Positionnez le premier point, en cliquant sur l'emplacement du départ
  • Une fois le point de départ positionné, ne s'afficheront en bleu plus que les chemins qui passent par le dernier point
  • Cliquez alors sur un point d'un chemin en bleu pour ajouter une section allant jusqu'à ce point dans le parcours.
  • Lorsque vous voulez passer par un embranchement pour changer de chemin, cliquez d'abord sur le point de jonction entre le chemin en cours et le chemin à prendre, pour pouvoir ensuite continuer le parcours sur le nouveau chemin

Positionner des points libres

Les chemins Open Street Map vous permettent de créer facilement et rapidement un parcours en utilisant les chemins connus.
Mais il peut arriver que le chemin que vous souhaitez prendre soit absent, ou qu'il manque une liaison. Vous pouvez alors positionner des points libres sur la carte. Pour cela, utilisez le menu "Point libre".
Vous pouvez ensuite repasser en mode "chemins Open Street Map", en utilisant le menu "Suivre chemins".

Enregistrer le parcours

Une fois le parcours terminé, cliquez sur "Enregistrer", entrez le nom du parcours et la collection dans laquelle vous souhaitez enregistrer ce parcours.
