Les Collections

Les collections vous permettent d'organiser vos parcours en "répertoires".
Par exemple:

  • Mes Parcours → les parcours que vous avez fait
  • À Faire → ceux que vous souhaitez faire
  • 🌴Voyages → parcours pendant vos voyages

Chaque collection vous permet de voir une carte avec tous les parcours de cette collection, de les consulter, trier, filtrer, rechercher...

Le menu "Toutes les collections" permet d'avoir la même vue en regroupant tous les parcours de vos collections.

Vous pouvez également organiser les parcours au sein d'une même collection, en utilisant les tags (voir page sur les tags TODO lien)

Mes Parcours

La collection "Mes Parcours" est automatiquement créée à la création de votre compte.

  • Elle appraît toujours en premier,
  • Lorsque vous créer une trace avec votre GPS (TODO lien), la trace est automatiquement enregistrée dans cette collection (mais vous pouvez ensuite les déplacer TODO lien)
  • Elle ne peut pas être supprimée (car c'est là que vont vos traces par défaut),
  • Mais vous pouvez la renommer.

Gérer les collections

Créer une collection

Pour créer une collection:

  • Dans le menu principal (☰), cliquez sur "Nouvelle collection..."
  • Donnez lui un nom, puis validez

Renommer une collection

Pour renommer une collection, utilisez le menu "Renommer", accessible par différents moyens:

  • Soit depuis le menu principal (☰), en cliquant sur les 3 points verticaux (⋮) en face de la collection
  • Soit lorsque vous êtes sur la collection:
    • en utilisant les 3 points verticaux (⋮) à côté du nom en haut de l'écran
    • en double-cliquant, ou avec un appui long, sur le nom de la collection en haut de l'écran

Supprimer une collection

Pour supprimer une collection, utilisez le menu "Supprimer":

  • Soit depuis le menu principal (☰), en cliquant sur les 3 points verticaux (⋮) en face de la collection
  • Soit lorsque vous êtes sur la collection, en utilisant les 3 points verticaux (⋮) à côté du nom en haut de l'écran
⚠ Attention ! Une fois supprimée, TOUT le contenu de la collection est perdu.
