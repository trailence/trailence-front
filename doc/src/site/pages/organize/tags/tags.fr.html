Les tags

Les tags vous permettent d'organiser vos parcours au sein d'une collection, en attribuant leur attribuant des "étiquettes".
Par exemple, vous pouvez créer des tags:

  • Avec le nom des personnes qui étaient avec vous → permet de retrouver facilement les parcours effectués avec une personne, et de les partager avec elle
  • "bivouac" → retrouvez tous vos parcours de bivouac
  • "urbain" → les parcours de visite de villes
  • "itinérance" → les parcours sur plusieurs jours
  • etc...

Un parcours peut avoir plusieurs tags.
Les tags appartiennent à une collection : chaque collection possède donc sa propre liste de tags.

Editer les tags d'une collection

Dans le menu de la collection, cliquez sur "Éditer les tags".
Une fenêtre s'affiche et vous permet de créer de nouveaux tags, renommer les tags existants ou de les supprimer.

Affecter des tags à un ou des parcours

  • Depuis la page d'un parcours, cliquez sur la section Tags, juste au-dessus de la description
  • Depuis le menu d'un parcours, cloquez sur "Tags"
  • Depuis la liste des parcours d'une collection, sélectionnez plusieurs parcours, puis cliquez sur "Tags"
Une fenêtre s'affiche et vous permet de sélectionner ou désélectionner des tags pour le(s) parcours.
Depuis cette fenêtre, vous pouvez également ajouter de nouveaux tags, ou les modifier en cliquant sur "Modifier" en bas à gauche.
