Les tags

Les tags vous permettent d'organiser vos parcours au sein d'une collection, en attribuant des "étiquettes" à vos parcours.
Par exemple, vous pouvez créer des tags:

  • Avec le nom des personnes qui étaient avec vous → permet de retrouver facilement les parcours avec une personne, et de partager ces parcours avec cette personne
  • "bivouac" → retrouvez tous vos parcours de bivouac
  • "urbain" → les parcours de visite de villes
  • "itinérance" → les parcours sur plusieurs jours
  • etc...

Un parcours peut avoir plusieurs tags.
