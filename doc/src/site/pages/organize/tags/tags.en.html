Tags

Tags let you organize your routes within a collection by assigning them labels.
For example, you can create tags:

  • With the names of the people who were with you → makes it easy to find routes with a specific person and share them with them
  • "bivouac" → find all your bivouac routes
  • "urban" → routes for city visit
  • "multi-day" → routes lasting several days
  • etc...

A route can have multiple tags.
Tags belong to a collection, so each collection has its own list of tags.

Edit collection tags

In the collection menu, click on “Edit tags”.
A window opens allowing you to create new tags, rename existing ones, or delete tags.

Assign tags to one or more routes

  • From a trail page, click on the Tags section just above the description
  • From the menu, click on “Tags”
  • From a collection’s list, select multiple routes, then click on “Tags”
A window opens allowing you to select or deselect tags for the selected route(s).
From this window, you can also add new tags or edit them by clicking on “Edit” in the bottom-left corner.
