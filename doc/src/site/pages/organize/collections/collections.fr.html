Les Collections

Les collections vous permettent d'organiser vos parcours en "répertoires".
Par exemple:

  • Mes Parcours → les parcours que vous avez faits
  • À Faire → ceux que vous souhaitez faire
  • 🌴Voyages → parcours pendant vos voyages
  • ...

Chaque collection vous permet de voir une carte avec tous ses parcours, de les consulter, trier, filtrer, rechercher...

Pour organiser vos parcours au sein d'une même collection, vous pouvez utiliser les tags.

Mes Parcours

La collection "Mes Parcours" est automatiquement créée à la création de votre compte.

  • Elle apparaît toujours en premier,
  • Lorsque vous créez une trace avec votre GPS, la trace est automatiquement enregistrée dans cette collection (mais vous pouvez ensuite les déplacer)
  • Elle ne peut pas être supprimée (car c'est là que vont vos traces par défaut), mais vous pouvez la renommer.

Toutes les collections

Le menu "Toutes les collections" permet d'avoir la même vue en regroupant tous les parcours de vos collections.

Copier / Déplacer

  • Le menu "Copier dans..." vous permet de copier un ou plusieurs parcours dans une autre collection
  • Le menu "Déplacer vers..." vous permet de déplacer un ou plusieurs parcours vers une autre collection
