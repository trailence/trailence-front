Modifier une trace

Pour commencer l'édition, allez sur le parcours, puis sur la carte cliquez sur "Éditer" en haut à droite.

L'édition vous permet de:

  • Modifier les étapes
  • Modifier l'altitude
  • Modifier le chemin
En utilisant les menus en haut de la carte.

Ajouter / supprimer / modifier une étape

Pour ajouter une étape, sélectionnez un point du parcours, puis le menu "Etape" et "Créer une étape".
Pour modifier ou supprimer une étape, sélectionner le point de l'étape sur la carte, ou, plus facilement, cliquez sur l'étape dans la liste des étapes. Allez ensuite dans le menu "Etape" pour modifier, déplacer ou supprimer l'étape sélectionnée.

Altitude

Différentes options vous permettent de modifier ou améliorer les altitudes.

  • Lorsque vous sélectionnez un point du parcours, vous pouvez modifier l'altitude manuellement dans l'encadré "Sélection".
  • En sélectionnant une section du parcours, le menu "Altitude" vous propose de nombreuses options pour modifier l'altitude de la zone sélectionnée.

Compléter le parcours

Vous pouvez compléter le parcours, en sélectionnant le point de départ, ou le point d'arrivée, puis en allant dans le menu "Chemin", des options vous propose de compléter le parcours.

Et plus...

De nombreuses options sont disponibles pour l'édition de la trace, n'hésitez pas à parcourir les menu, à sélectionner un point ou une zone puis consulter à nouveau les menus proposés pour la sélection.
